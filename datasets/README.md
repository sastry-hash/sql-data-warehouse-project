# Datasets

This folder contains raw source datasets used in the data warehouse project.

These files represent the Bronze layer inputs and are loaded into
the database before applying Silver and Gold transformations.

## Files
- cust_info.csv: Customer master data
- prd_info.csv: Product information
- sales_details.csv: Sales transaction details
- PX_CAT_G1V2.csv: Product category reference data
- CUST_AZ12.csv: ERP customer attributes
- LOC_A101.csv: Customer location data
